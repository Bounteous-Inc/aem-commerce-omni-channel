5.2       Setup Tasks for AEM Mobile for AEM 6.2  (ALREADY DONE)

 

1.       In your AEM instance, go to Tools > Web Console. A new tab or window is opened.

¥    Scroll down to ÒAdobe Digital Publishing Solution ClientÓ and click the edit icon (pencil).

¥    Enter the IMS Client ID and Client Secret in the ÒDPS Client IdÓ and ÒDPS Client SecretÓ fields and click Save. You can then close the Web Console tab or window.

 

 

 

2.       Back on Main AEM page, go to Tools > Operations > Cloud > Cloud Services.

¥    Scroll down to ÒAdobe Digital Publishing SolutionÓ (green icon, under ÒAdobe Creative CloudÓ).

¥    Click ÒConfigure Now.Ó

¥    In the ÒCreate ConfigurationÓ pop-up window, enter a Title and Name, and click Create (Name should have no spaces).

¥    In the Adobe Digital Publishing Solution Configuration pop-up window, enter the Device Id and Device Token from the IMS Device Token Generator used earlier.

¥    Click ÒTest DPS SettingsÓ to check this. If the test is not successful, send email to DL-AEM           Mobile TechTalk for troubleshooting.

¥    Click OK, and then navigate back to the Main AEM window.
